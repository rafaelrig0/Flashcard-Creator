# Flashcards App

Aplicação web para criação e revisão de flashcards, organizada por pastas.
O objetivo do projeto é facilitar o processo de estudo através de revisões rápidas e acompanhamento de desempenho.

# Funcionalidades

📁 Criar e visualizar pastas de estudo

🃏 Criar flashcards dentro das pastas

🔁 Realizar revisões dos cards

📊 Visualizar taxa de acerto por pasta

⚡ API construída com FastAPI

💾 Banco de dados PostgreSQL

# Tecnologias Utilizadas

PostgreSQL
